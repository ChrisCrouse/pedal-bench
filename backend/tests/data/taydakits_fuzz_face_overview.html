Fuzz Face Step 1 of 5

This is the Fuzz Face, the legendary fuzz pedal. With this PCB you can build the original or many other variants using either PNP or NPN transistors. The PCB includes a dedicated pad for a trimmer, allowing precise adjustment of the circuit’s behavior.
